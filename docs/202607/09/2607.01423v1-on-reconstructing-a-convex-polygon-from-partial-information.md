## 摘要
重建问题要求构造一个（凸）多边形，使其具有指定的特征集，例如有序的边长集或有序的多边形内角集。本文系统地探索了在指定一组或两组特征的所有场景下的重建问题。其中一些场景已有深入研究，另一些我们开发了测试算法和/或难解性结果，还有许多引发了有趣的开放问题供未来研究。

## Abstract
The reconstruction problem asks to construct a (convex) polygon that has a specified set of features, such as an ordered set of edge-lengths or an ordered set of polygon-angles. In this paper, we do a systematic exploration of the reconstruction problem in all scenarios where one or two sets of features have been specified. Some of these scenarios were well-studied already, for some we develop testing-algorithms and/or hardness results, and many give rise to interesting open problems for future study.

  - **极性对偶**：对包含原点的凸多边形，角向量与边法向量互换，角距离与边距离互为倒数。用于将“角向量+边距离”等场景等价转换。
  - **传播公式**：从一个未知变量（如角距离 `x=d0`）开始，利用给定特征递推其他所有未知量，并用闭合条件（如 `d_n = d_0`）消元：

### 6. 论文的主要结论与发现
- **单特征场景**：
  - 只给出三角形面积：总存在凸实现，可线性时间构造（RealRAM）。
  - 只给出边距离：总存在星形实现；凸实现的充要条件可在线性时间（边距离互异时）测试。
  - 只给出角距离：总存在星形实现；凸实现可通过极性与边距离情况判断。
  - 边长、角向量、边法向量场景均有已知充要条件或线性时间测试。
- **双特征场景**：
  - **平凡/已知**：边法向量+边长（Minkowski）、边法向量+面积（L0-Minkowski，至今无构造算法）、角向量+角距离、边法向量+边距离、边长+角距离。

  - **难解性**：角向量+面积（无标签）NP-hard；角距离+边距离（RealRAM 归约到 2-划分）；边长+角向量（圆柱上）NP-hard。
  - **开放问题**：L0-Minkowski 构造算法、面积+边距离、角向量+边距离等。
