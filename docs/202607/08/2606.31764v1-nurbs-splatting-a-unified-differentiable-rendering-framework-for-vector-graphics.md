## 摘要
平面有理样条的可微绘制仍然在很大程度上未被探索，尽管它们在矢量图形和设计中广泛应用。现有的可微矢量化渲染器主要关注贝塞尔曲线，并依赖解析光栅化，这可能导致梯度不稳定和灵活性受限。我们提出了 NURBS 抛雪球，这是一个将平面有理曲线表示为连续高斯场的统一框架。通过沿曲线参数域和闭合区域内采样高斯函数，渲染被重新表述为一个具有稳定梯度的平滑累积过程。我们的方法自然支持长样条、有理权重、非均匀节点以及闭合区域填充。我们展示了它在书法重建、矢量化框架和长样条图像抽象方面的有效性，显示出相较于现有方法的更高稳定性和重建质量。

## Abstract
Differentiable rendering of planar rational splines remains largely underexplored, despite their widespread use in vector graphics and design. Existing differentiable vector renderers primarily focus on Bézier curves and rely on analytic rasterization, which can suffer from gradient instability and limited flexibility. We propose NURBS Splatting, a unified framework that represents planar rational curves as continuous Gaussian fields. By sampling Gaussians along the curve parameter domain and inside closed regions, rendering is reformulated as a smooth accumulation process with stable gradients. Our method naturally supports long splines, rational weights, non-uniform knots, and closed-region filling. We demonstrate its effectiveness in calligraphy reconstruction, vectorization frameworks, and long-spline image abstraction, showing improved stability and reconstruction quality over existing approaches.

### 2. 论文提出的方法论

- **核心思想**:
  - 将NURBS曲线及其填充区域重新表述为**连续的2D高斯场**。通过在这种高斯场上进行采样，将离散的、可能导致梯度不稳定的解析渲染过程，转化为平滑的、具有稳定梯度的累积过程。

- **关键技术细节与流程**:
  1.  **NURBS基元定义 (NURBS Primitives)**:
      - 曲线由可学习的**控制点**、**有理权重**和由**内部节点间隔**重建的**节点向量**定义。为便于优化，关键点作为主要可学习参数，并据此推导控制点和节点向量。
      - 支持开放曲线和闭合曲线的不同参数化方式。每条曲线还包含可学习的颜色、透明度，并拓展控制点到3D以编码局部**描边宽度**。
  
  2.  **轮廓上的高斯采样 (Gaussian Sampling on Contours)**:
      - 沿曲线**自适应采样**各向同性的高斯基元，采样密度与曲线的弧长成正比。
      - 通过Cox-de Boor递推的局部支撑性质，将评估成本从`O(Mn)`降至`O(Mp)`（其中M为采样点数量，n为控制点数量，p为曲线阶数），显著提高计算效率。
      - 高斯的尺度仅由局部描边宽度决定，方向固定为各向同性，避免了类似Bézier Splatting中因估计切线旋转而产生的模糊或尖峰伪影。
  
  3.  **区域内部的高斯填充 (Gaussian Sampling in Regions)**:
      - 对于闭合曲线，用一个**可微的符号距离场（SDF）**来调制内部高斯基元的透明度。
      - 在边界框内生成均匀2D网格点，其透明度由基于SDF的sigmoid软掩膜决定（SDF在区域内为正值）。
      - 提出**网格步长退火策略**：在优化初期使用粗网格以加速收敛，后期精细网格以保留细节。SDF的计算与边界点相关，使得整个过程可微。

  4.  **可微抛雪球渲染 (Differentiable Splatting Rendering)**:
      - 利用GaussianImage的基于瓦片的光栅化器，对所有采样的2D高斯按照深度顺序从前到后进行Alpha混合，合成最终图像。

  5.  **优化目标 (Optimization)**:
      - **总损失**: `L = L_img + λ_geo * L_geo`，其中`L_img`是特定任务的图像重建损失（如MSE, 感知损失）。
      - **几何正则化**:
        - **导数损失 (`L_deriv`)**: 惩罚曲线的高阶导数（默认3阶）的平方模长，即经典的曲线光顺能量，防止曲线过于曲折。
        - **包围盒损失 (`L_bbox`)**: 通过软惩罚项防止曲线控制点漂移到图像区域之外。

### 6. 论文的主要结论与发现

- **首创性**: 成功构建了第一个能够联合优化NURBS曲线的控制点、有理权重和非均匀节点向量的2D图像空间可微渲染框架。
- **性能提升**: 在书法重建任务中，在多个指标上显著优于基于多项式B样条的基线，尤其在带有尖锐弯曲和钩子的日语假名字符上，PSNR提升高达+2.10 dB。
- **效率优势**: 高斯抛雪球的渲染管道使优化更快（书法重建任务中速度提升1.2倍）。当控制点数量增加时，避免了B样条转贝塞尔的矩阵乘法开销，展现出更好的可扩展性。
- **质量改进**: 各向同性的高斯采样策略天生避免了Bézier Splatting方法在粗笔触、高曲率等区域出现的模糊或尖峰伪影。
- **NURBS特性的价值**: 消融实验证明，有理权重和非均匀节点提供了互补的几何灵活性。“有理权重”的去除会带来最显著的质量下降（-0.69dB PSNR），而“非均匀节点”的去除则损失最大速度优势。

  - **可微性良好**: 整个pipeline基于采样和场函数，梯度计算平滑而稳定，优于基于解析求交的方法。

### 8. 不足与局限

- **方法局限**:
  - **区域填充效率**: 当前的区域填充策略依赖于基于启发式SDF的网格，论文承认，对于规则几何形状，其优势相较于扫描线方法有限。
  - **内存占用**: 对于高度复杂的插图，高斯基元的数量会随着曲线复杂度增长，可能导致内存使用量增加。
- **实验局限**:

  - **与最先进方法的直接对比**: 未能与当前性能最优的“Bézier Splatting (BS)”方法在同一个完整任务（如书法重建或矢量化）下进行直接的定量对比，仅进行了诊断性对比和基于原型的间接对比，其性能的绝对优越性有待更多检验。
- **应用限制**: 当前框架仅处理2D曲线。对3D NURBS（如NURBS曲面）的可微渲染是论文指出的未来工作方向。
